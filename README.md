# 🏢 Portal Atlantis - Gestão Condominial

O **Portal Atlantis** é um sistema de gestão para condomínios desenvolvido em Django. Ele foi criado para facilitar a comunicação entre síndicos, porteiros e moradores, centralizando informações de acesso, avisos e cadastros.

---

## 🚀 Funcionalidades Atuais
- **Tela de Login Personalizada:** Acesso seguro com interface escura e dourada.
- **Dashboard Administrativo:** Painel visual para gestão rápida do condomínio.
- **Cadastro de Moradores:** Fluxo dedicado para novos usuários solicitarem acesso.
- **Gestão de Portaria:** Visualização de moradores por unidade (Bloco/Casa).

---

## 🛠️ Tecnologias Utilizadas
- **Python 3.x**
- **Django Framework**
- **Bootstrap 5** (Interface)
- **SQLite** (Banco de dados de desenvolvimento)
